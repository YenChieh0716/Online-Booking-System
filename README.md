# Online-Booking-System
## 安裝套件
* npm install semantic-ui-react semantic-ui-css react-router-dom 
* 引用導覽列 navbar.js和header.js一起include
* 登入註冊頁面在Signin.js
* 因有更改到一些css，記得複製'semantic-ui-css/semantic.min.css'內容並覆蓋原本檔案內容;
